# 👋 Shopping list
 
A shopping list item app to enable users view and add list items. 

## 🚀 Features

- **View list items**: Users are able to view all shopping list items.
- **Add List Items**: Users are able to add shopping list items.

## 🛠️ Technologies

- `HTML`
- `CSS`
- `Javascript`
